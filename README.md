# Installation
1. Download the source code from the `Releases`
2. Open `chrome://extensions`
3. Select `Load unpacked`
4. Select the downloaded source code

# Storage

Type : Chrome local storage API

Stores :

- `Unix Epoch (ms)` : Milliseconds between the time that stopwatch was started to January 1, 1970 00:00
- `Elapsed Time (ms)` : Milliseconds between now to `Unix Epoch`

# Tools used

- npm + React + Vite(Rolldown)
- TailwindCSS
